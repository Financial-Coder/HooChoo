# 후추 패밀리 앱 – 화면 와이어프레임 문서

## 1. 핵심 화면 정의
### 1.1 인증
- **로그인 화면**: 이메일/비밀번호 입력, "가족 초대 받지 못했나요?" 안내 텍스트. 둥근 입력 필드와 고양이 아이콘이 포함된 로그인 버튼.
- **회원가입 화면(관리자 한정)**: 관리자 전용 초대 코드 입력 후 이름/이메일/비밀번호 생성. 가족 구성원 초대용 링크 복사 액션 포함.
- **초대 가입 화면**: 초대 링크 진입 시 이름/닉네임, 비밀번호 설정. 가입 완료 후 즉시 메인 피드 이동.

### 1.2 메인 피드
- **피드 화면**: 상단에는 후추 프로필 요약(대표 이미지, 생일, 한 줄 소개). 이어서 최신순 카드 리스트. 각 카드에 미디어 썸네일/영상 플레이어, 캡션, 좋아요/댓글 수. 하단에 업로드 플로팅 버튼.
- **무한 스크롤**: 6~8개씩 로드, 스크롤 최하단에서 자동 다음 페이지 호출. 로딩 스켈레톤에 고양이 발바닥 패턴 사용.

### 1.3 업로드 플로우
- **업로드 모달(혹은 전용 화면)**: 이미지·영상 탭 선택 → 파일 드롭존 → 캡션 입력 → 업로드 진행률 표시. 용량 초과 시 경고 배지.
- **썸네일/영상 처리 상태**: 업로드 완료 후 “후추에게 전달 중…” 토스트, 처리 완료 시 피드 상단에 자동 추가.

### 1.4 상세/댓글
- **포스트 상세 시트**: 피드 카드 탭 시 전체화면 모달. 상단에 미디어, 우측 상단 닫기. 하단에는 댓글 리스트, 이모지 빠른 입력, 수정/삭제 아이콘(작성자만 활성화).
- **댓글 입력 영역**: 둥근 입력창 + 🐾 버튼(자주 쓰는 이모지 단축) + 실시간 반영 스피너.

### 1.5 후추 프로필
- **프로필 화면**: 대표 이미지 + 이름/생일. "후추 레벨" 같은 재미 요소 고려. 가족 구성원 조각 아이콘으로 참여도 표현. 프로필 편집은 관리자 전용.

### 1.6 추억 필터
- **필터 패널**: 상단 탭/슬라이드 시트. 연도 드롭다운, 타입 토글(사진/영상), “오늘의 후추” 버튼(랜덤 한 장 풀스크린 미리보기).

## 2. 사용자 흐름
1. **로그인 플로우**: 앱 진입 → 로그인 → JWT 세션 확보 → 메인 피드 로드.
2. **관리자 초대 플로우**: 관리자 로그인 → `가족 초대` 버튼 → 이메일 전송 or 링크 복사 → 가족이 링크로 가입.
3. **콘텐츠 업로드**: 피드 FAB 클릭 → 업로드 모달 → 파일 업로드 + 캡션 입력 → 업로드 성공 시 피드 최상단 삽입, 썸네일 생성 상태 표시.
4. **피드 상호작용**: 사용자가 스크롤 → 무한 스크롤 로딩 → 특정 카드 좋아요 클릭(하트 애니메이션) → 댓글 버튼으로 상세 시트 열기 → 댓글 작성/수정/삭제.
5. **추억 필터**: 필터 버튼 클릭 → 조건 선택 → 피드가 조건에 맞게 재호출 → “오늘의 후추” 누르면 랜덤 미디어 풀스크린 뷰 후 피드 복귀.

## 3. 화면별 레이아웃 가이드
- **헤더**: 좌측 후추 로고, 우측 가족 메뉴(프로필, 로그아웃). 모바일에서는 햄버거 메뉴로 축소.
- **색상/스타일**: 배경 파스텔 베이지, 주요 버튼 파스텔 오렌지, 포인트 색상 라벤더. 모든 버튼과 카드 모서리 16px 이상. 고양이 아이콘(귀, 발바닥)을 섹션 구분 요소로 사용.
- **피드 카드**: 카드 상단 미디어, 하단 텍스트 영역. 좋아요 버튼은 하트 아이콘 + 카운트, 댓글 버튼은 말풍선 + 카운트. 영상은 중앙 플레이 버튼 오버레이.
- **모바일 우선**: 360px 기준으로 단일 컬럼, 데스크톱에서는 2열 그리드 + 우측에 추억 필터 고정 패널.
- **업로드 모달**: 상단 단계 인디케이터(파일 선택 → 캡션 → 확인). 파일 드롭존 안에 고양이 실루엣 + “후추의 새 사진을 올려주세요” 카피.

## 4. 동적/실시간 UX 메모
- **무한 스크롤 로딩 상태**: 카드 스켈레톤을 2~3개 표시, 로딩 끝나면 부드러운 페이드.
- **실시간 댓글**: WebSocket/SSE로 새로운 댓글 푸시, 입력 시 Optimistic UI 적용 후 실패 시 토스트로 롤백 안내.
- **좋아요 토글**: 즉시 하트 애니메이션+카운트 변경, 서버 응답 실패 시 원상복구.
- **업로드 진행 표시**: 파일 업로드 중에는 진행률 바와 취소 버튼 표시, 썸네일 처리 중에는 “썸네일 만드는 중…” 배지.
- **오늘의 후추**: 버튼 클릭 시 랜덤 미디어 풀스크린을 5초간 보여주고 자동으로 닫히거나, 드래그 다운으로 닫기.

## 5. 향후 디자인 전개 메모
- 다크모드/고대비 테마 고려.
- 가족별 커스텀 스티커 위치(댓글 말풍선, 피드 카드 상단 등) 확보.
- 반응형 폰트 스케일 정의(모바일 16px, 데스크탑 18px 기본).
